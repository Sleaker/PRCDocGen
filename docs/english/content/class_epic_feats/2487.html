:: Epic Ninja ::

You are the ultimate ninja with legendary skills and a body count to match.

Hit Die: d6
Skill Points at Each Additional Level: 8+ Int Modifier
Bonus Feats: The epic ninja gains a bonus feat every four levels after 20th
Special:
Sneak Attack: this continues to improve by 1d6 at each odd-numbered level the epic ninja gains.

Epic Ninja Bonus Feats List:
Blinding Speed, Crippling Strike, Defensive Roll, Epic Skill Focus, Epic Dodge, Epic Reputation, Improved Evasion, Improved Sneak Attack, Opportunist, Self-Concealment, Skill Mastery, Slippery Mind, Superior Initiative.
