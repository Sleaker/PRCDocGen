:: Paragon Visionary ::

Type of Feat: Epic
Prerequisite: Spellcraft 24 ranks, dark foresight, devil's sight, see the unseen, voidsense.
Specifics: You can see through illusions, magical darkness, shapechangers, and other effects (as true seeing). This is a supernatural ability.

You receive a bonus on all Listen, Sense Motive, and Spot checks equal to double your Wisdom modifier or +6, whichever is greater. This is an extraordinary ability.

Use: Automatic.
