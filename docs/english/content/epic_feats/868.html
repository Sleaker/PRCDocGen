:: Improved Whirlwind Attack ::

Type of Feat: Combat
Prerequisite: 21st level, Intelligence 13+, Dexterity 23+, Dodge, Expertise, Mobility, Spring Attack, and Whirlwind Attack or Bladewind class ability.
Specifics: The character performs a full attack action, and makes one melee attack at full base attack bonus against all opponents they threaten.
Use: Selected.

Requires All of These Feats:

Expertise
Spring Attack

Requires One of These Feats:

Bladewind
Whirlwind Attack
