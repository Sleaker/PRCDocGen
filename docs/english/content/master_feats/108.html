:: Divine Song III ::


Type of Feat: Class Specific
Prerequisite: Favored of Milil
Specifics: The Favored of Milil can sing a song which behaves as a clerical spell with a wide radius. Each song is limited to three uses per day, unless the Favored of Milil has the Greater Divine Song feat, which will allow each song to be used five times per day.
Use: Selected
