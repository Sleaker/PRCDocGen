:: Mastery of Elements ::

Type of Feat: Class Specific
Prerequisite: Archmage Level 1
Specifics: The Archmage can alter an arcane spell when cast so that it utilizes a different element from the one it normally uses. This ability can only alter a spell with the acid, cold, fire, electricity, or sonic descriptor.
Use: Selected.

Subradial entries:

Mastery of Elements(Cold)
Mastery of Elements(Electricity)
Mastery of Elements(Fire)
