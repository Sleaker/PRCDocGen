:: Superior Dominate ::

Type of Feat: Class Specific
Prerequisite: Thrallherd level 9.
Specifics: The Thrallherd no longer has to pay 4 additional power points for a Psionic Dominate to affect aberrations, dragons, elementals or outsiders. This does not increases the save DC as if the Thrallherd had spent the power points.
Use: Automatic.
