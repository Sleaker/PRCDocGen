:: Sun Domain Powers ::

Type of Feat: Domain
Prerequisite: Access to the Sun domain.
Granted Power: Once per day, you can perform a greater turning. The greater turning is like a normal turning attempt, except that undead creatures that would be turned are destroyed instead.
Bonus Spells: The cleric gains access to the following spells at the specified spell level: Bless Weapon (1), Searing Light (2), Sunbeam (7), Sunburst (8)
Use: Automatic.

Required For the Following Feats:

Light of Aurifar
