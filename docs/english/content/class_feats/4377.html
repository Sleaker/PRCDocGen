:: Rapid Blast ::

Type of Feat: Class
Prerequisite: Spellfire Channeler level 4.
Specifics: At 4th level, the spellfire channeler can release two blasts of spellfire as a standard action. This increases to three blasts per standard action at 8th level. Each blast after the first suffers a cumulative -2 penalty to attack rolls (-0 for the first blast, -2 for the second, and -4 for the third).
Use: Selected.
