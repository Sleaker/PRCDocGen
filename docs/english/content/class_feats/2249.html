:: Blightmind ::

Type of Feat: Class Specific
Prerequisite: Talontar Blightlord 4.
Specifics: Once per day, the Blightlord gains the ability to dominate a blightspawned creature. The creature must make a saving throw vs 10 + Blightlord level + Wisdom modifier or be dominated for 1 minute per Blightlord level.
Use: Selected.
