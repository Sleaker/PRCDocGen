:: Researched Epic Spell: Night's Undoing ::

School: Evocation [Light]
Components: V,S
Range: Touch
Target: One touched creature or object
Area: 40-ft.-radius
Duration: Permanent
Saving Throw: None
Spell Resistance: Yes

Nights Undoing bestows an ever-shining glow of light on the target. It creates a shadow-negating brightness that lights up the area, instantly dispels all darkness effects within its area.
