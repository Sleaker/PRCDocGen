:: Trickery Domain Powers ::

Type of Feat: Domain
Prerequisite: Access to the Trickery domain.
Granted Power: Bonus to Hide, Persuade, Search, Disable Trap, Move Silently, Open Lock, and Pick Pockets checks equal to 1 per 2 levels. This effect lasts for 5 turns + the cleric's Charisma modifier.
Bonus Spells: The cleric gains access to the following spells at the specified spell level: Invisibility (2), Invisibility Sphere (3), Mirror Image (4), Improved Invisibility (5).
Use: Automatic.

Required For the Following Feats:

Divine Intercession
