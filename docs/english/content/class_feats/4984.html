:: Adept Shaped Breath ::

Type of Feat: Breath Effect
Prerequisite: Dragonfire Adept level 5

The first four non-hostile targets caught in your breath are not affected by it.
You can apply any one other breath effect you know to this breath. If used with Cloud Breath, you can only use the fire breaths. Any other breath used with both active will apply this effect only.
Use: Selected
