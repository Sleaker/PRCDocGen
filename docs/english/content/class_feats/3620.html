:: Multiple Throw ::

Type of Feat: Class
Prerequisites: Soulknife 17th
Specifics: At 17th level and higher, a soulknife can throw a number of mind blades per round equal to the number of melee attacks he could make. Thus, a 17th-level soulknife could throw two mind blades, with a base attack bonus of +12/+7.
Use: Automatic
