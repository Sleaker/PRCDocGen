:: Bolt of Fire ::

Type of Feat: Class Specific
Prerequisite: Pyrokineticist level 3
Specifics: Starting at 3rd level, as a standard action, a pyrokineticist can launch a bolt of psionically manifested fire at any target in line of sight within 60 feet. This effect is treated as a ranged touch attack and deals 1d6 points of fire damage for each class level the pyro has.
Use: Selected
