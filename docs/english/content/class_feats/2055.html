:: Cast Domain Level 7 ::

Type of Feat: Domain
Prerequisite: Any Bonus Domain
Specifics: This feat allows the user to cast spells from his bonus domains. To cast a spell, select the slot and level you wish to cast, and click on the target. You must have at least one spell memorized of the appropriate level for the spell to cast. Using the Check Domain Slots feat will tell you which Domain is in which slot.
Use: Selected.
