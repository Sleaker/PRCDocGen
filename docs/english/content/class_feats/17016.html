:: Warmage Continual Flame ::

Caster Level(s): Cleric 3, Wizard / Sorcerer 2
Innate Level: 3
School: Illusion
Descriptor(s):
Component(s): Verbal, Somatic
Range: Touch
Area of Effect / Target: Single Creature or Item
Duration: Permanent
Additional Counter Spells:
Save: None
Spell Resistance: No
This creates a magical flame that burns as bright as a torch until dispelled. It can be cast on any item that can be equipped to create a permanent source of light.
