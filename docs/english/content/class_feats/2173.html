:: Extend Utterance ::

Type of Feat: General
Prerequisite: Truespeak 6 ranks, Able to speak utterances.
Specifics: You can speak an extended utterance. An extended utterance lasts twice as long as normal. A utterance with a duration of concentration, instantaneous, or permanent is not affected by this feat.
Using this feat increases the DC to speak the utterance by 5.
Use: Selected.
