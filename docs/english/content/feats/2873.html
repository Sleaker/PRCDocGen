:: Exceptional Artisan II ::

Type of Feat: General
Prerequisite: Any item creation feat.
Specifics: You are an expert at creating magic items faster than usual. When determining the time you need to craft any item, reduce the base time by 25%. You can gain this feat multiple times. Its effects stack (geometrically).
Usage: Automatic.

Requires All of These Feats:

Exceptional Artisan I

Feat Replaced By:

Exceptional Artisan III

Required For the Following Feats:

Exceptional Artisan III
