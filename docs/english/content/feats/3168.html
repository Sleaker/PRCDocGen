:: Exalted Turning ::

Type of Feat: Exalted.
Prerequisite: Turn Undead, Good or Neutral Alignment*.
Specifics: Any Undead that you turns take an extra 3d6 damage in addition to the normal turning effect.
Use: Automatic.

(*Note: You can select this feat even when evil, but it will not give you any benefits, so be smart and don't choose it if you can't use it.)

Requires All of These Feats:

Turn Undead
